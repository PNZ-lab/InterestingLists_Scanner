# InterestingLists_Scanner.py:
This script has access to 141 dataframes from across Prof. Ntziachristos research as well as publicly available data.
This script scans for substantial and significant differential expression, splicing, and proteomics for a given set of genes. 

All results are plotted and can be saved to a pdf file. Example:
